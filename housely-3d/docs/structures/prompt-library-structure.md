# Prompt Library Structure

## Purpose

Define how prompts are categorized, versioned, and reviewed so AI behavior stays intentional.

## Structure

- System prompts.
- Task prompts.
- Vision prompts.
- Domain prompts.
- Prompt experiments and archived variants.

## Dependencies

- AI documentation plan.
- Research documentation structure.
- Decision logs.

## Review Checklist

- Is the prompt purpose obvious.
- Is the version history clear.
- Are fallbacks and constraints documented.

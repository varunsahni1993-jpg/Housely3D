# Research

This folder stores research notes, source material, and experimental findings.

## Rules

- Separate sources from conclusions.
- Record the question being answered before collecting evidence.
- Link research outcomes to decisions and prompts when they influence direction.

## Suggested Structure

- `sources/` - traceable references and source notes.
- `experiments/` - test runs and evaluation notes.
- `findings/` - summarized conclusions.
- `notes/` - working notes that can later be converted into formal research records.

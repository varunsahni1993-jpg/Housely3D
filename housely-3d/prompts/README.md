# Prompts

This folder governs the prompt library for Housely-3D.

## Rules

- Treat prompts as versioned product assets.
- Store canonical prompts separately from experimental variants.
- Link prompts to research findings, model choices, and release decisions.

## Suggested Structure

- `library/` - canonical prompt families and approved variants.
- `system/` - high-trust system prompts and instruction sets.
- `workflows/` - task prompts and reusable flows.
- `archive/` - retired prompts and historical variants.

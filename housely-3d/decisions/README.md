# Decisions

This folder stores ADRs and decision history for the platform.

## Rules

- Record every architectural or product choice that has lasting consequences.
- Keep decision IDs stable.
- Link decisions to research, plans, and implementation notes.

## Templates

- [ADR Template](../templates/adr-template.md)
- [Decision Log Template](../templates/decision-log-template.md)

## Suggested Structure

- `records/` - approved ADRs and decision logs.
